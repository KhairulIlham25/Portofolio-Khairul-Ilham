# Documents Folder

Folder ini untuk menyimpan file CV dan dokumen lainnya.

## Cara Upload CV:

1. **Siapkan file CV kamu dalam format PDF**
   - Nama file: `CV_Khairul_Ilham.pdf`
   - Pastikan ukuran file tidak terlalu besar (max 5MB)

2. **Taruh file PDF di folder ini:**
   ```
   assets/documents/CV_Khairul_Ilham.pdf
   ```

3. **Push ke GitHub:**
   ```bash
   git add assets/documents/CV_Khairul_Ilham.pdf
   git commit -m "docs: add CV PDF file"
   git push origin main
   ```

4. **Tombol Download CV akan otomatis bekerja!** ✨

---

**Note:** File `.pdf` sudah tidak di-ignore di `.gitignore`, jadi aman untuk di-push ke GitHub.
